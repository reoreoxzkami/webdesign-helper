# WebDesign Helper

WebDesign Helper は、初心者でも簡単にウェブページのデザインを編集できる Firefox 拡張機能です。

ページ上の要素をクリックするだけで、色・フォント・余白・サイズなどを直感的に変更できます。  
編集内容はリアルタイムで反映されるため、コードを書く必要はありません。

## 主な機能
- 要素をクリックして選択
- 色・背景色の変更
- フォントサイズ・太さの調整
- 余白 (margin / padding) の編集
- リアルタイムプレビュー
- シンプルで初心者に優しい UI
